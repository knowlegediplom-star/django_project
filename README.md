# Django API проект

## Запуск проекта

1. Клонировать репозиторий:
git clone https://github.com/knowlegediplom-star/django_project

2. Перейти в папку:
cd django_project

3. Запустить проект:
docker-compose up

4. Открыть в браузере:
http://127.0.0.1:8000/employees/

## Описание

В проекте реализован API для сотрудников с пагинацией, фильтрацией и CRUD-операциями.
